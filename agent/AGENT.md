## Agent
# Descipcion
- Proyecto donde busco crear una landing page academica dinamica con imagenes y ventanas modales interactivas con JavaScript vanilla, CSS 3 y HTML 5

# requerimentos del Proyecto:

- Crear un git ignore donde se ignore la carpeta agent para cualquier commit.
- En el documento HTML se requiere mostrar cinco imagenes envueltas en diferentes containers donde habra una pequeña del hito descripcion en formato parrafo. 
- En la ventana modal interactiva utilizar JavaScript y alargar la descripcion minimo en un texto de 100 palabras por cada hito.
- El titulo "Cinco hitos del pueblo vikingo" debe mostrarse encima de los containers.
- Debe aber modo oscuro/modo claro y guardar las preferencias en localstorage.
- Debe ser responsive
- Debe ser escalable. Es decir optimizado para crecer en un futuro.
- En el pie de pagina debe decir "Creado por: afrivillas".
- En el header debe decir "Los cinco hitos legendarios de los vikingos".
- Optimizado para CEO con google e indexable para buscadores de IA.
- Abajo de los cinco container en normas APA debe estar referenciado los recursos utilizados con su respoectivo link para verificar cualquier informacion.

# normas

- Toda informacion debe estar referenciada y debe ser veridica y verificable.
- No usar librerias externas solo JavaScript y CSS.
- Siempre pensar en escalabilidad a futuro es decir no repetior codigo, mejor modulariza y reutiliza.
